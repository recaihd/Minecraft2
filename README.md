## Como rodar o Projeto?

Se você ainda não tem o Python instalado na sua máquina, instale a versão [3.10.9](https://www.python.org/downloads/release/python-3109/) ou similar, pois a biblioteca ursina ainda não funciona nas versões mais atuais do python nem nas mais antigas

## Dependência do projeto

No seu vscode ou editor de código de preferência, abra o terminal e instale a seguinte dependência:

```bash
pip install ursina
```
## Execute o projeto com ```python <nomedoarquivo>.py```
